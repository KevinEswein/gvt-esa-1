Interaktive animierte Scheibe

Sie können die Scheibe durch Drücken der folgenden Tasten steuern:

  • L = Eine Drehung nach links (-15°)
  • R = Eine Drehung nach rechts (+15°)
  • A = Start / Stop kontinuierliches Drehen
  • M = Start / Stop Mario Animation
